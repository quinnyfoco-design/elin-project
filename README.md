# The elin project

The elin project focuses on a local and free coding models. 
It has multiple options such as Standard(runs on most
modern hardware), Cracked(Same) size as standard but abliterated,
and lite(Smaller than standard,
focused for less fast hardware or more speed on modern Hardware)
